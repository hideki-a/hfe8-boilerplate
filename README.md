# hfe8-boilerplate

[広島フロントエンド勉強会 Vol.8](https://hfe.connpass.com/event/52671/)に向けて、新しいBoilerplateを作成していきます。

## How To Use

1. `npm i`でパッケージをインストールします。
2. `npm start`でgulpのデフォルトタスクを実行します。
3. Enjoy Coding!
